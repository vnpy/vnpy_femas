## 说明

基于FEMAS期货柜台1.0.0版本开发的交易接口。

## 安装

安装需要基于2.7.0版本以上的[VN Studio](https://www.vnpy.com)。

直接使用pip命令：

```
pip install vnpy_femas
```

下载解压后在cmd中运行

```
python setup.py install
```
